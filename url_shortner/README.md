# shortCurtUrl

Projeto criado idealizado pela DIO com a orientação da Alexia Santos, consiste em um encurdador de Urls. Foi desenvolvido usando Node e Typecript, e o banco de dados MongoDB.

## Como rodar o projeto

- npm i
- npm run build
- npm run dev

## Bibliotecas

- Express
- Mongoose
- Nodemon
- Dotenv
